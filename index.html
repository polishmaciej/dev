About Me

My name is Maciek and I am an 18-year-old Web designer and developer.

With rich front-end expertise, I merge creativity and tech skills in each project. My goal is to craft visually appealing websites that function flawlessly for a memorable user experience.

Visually Appealing Websites

As a Frontend Developer, my mission is to create responsive and visually appealing user interfaces that are both functional and aesthetically pleasing.
